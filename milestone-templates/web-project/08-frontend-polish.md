# Milestone: Frontend Polish
> Type: milestone-template
> Category: web-project
> Order: 8
> Optional: false

## Goal
UI is responsive, accessible, performant, and consistent across browsers.

## Tasks
- [ ] Implement responsive design (mobile, tablet, desktop breakpoints)
- [ ] Test on major browsers (Chrome, Firefox, Safari, Edge)
- [ ] Run accessibility audit (WCAG 2.1 AA minimum)
  - [ ] Keyboard navigation works throughout
  - [ ] Screen reader compatible (alt text, ARIA labels, semantic HTML)
  - [ ] Colour contrast meets AA standards
- [ ] Optimise page load performance
  - [ ] Compress and lazy-load images
  - [ ] Minimise CSS/JS bundle size
  - [ ] Check Core Web Vitals (LCP, CLS, INP)
- [ ] Run /frontend-design for screenshot-driven comparison if matching a mockup
- [ ] Run /site-audit on deployed preview for comprehensive audit

## Notes
Use /frontend-design with a reference image for pixel-perfect matching. Use /site-audit for a full SEO, performance, accessibility, and security audit.
