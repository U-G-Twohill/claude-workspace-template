# Milestone: Client Handoff
> Type: milestone-template
> Category: web-project
> Order: 9
> Optional: true

## Goal
Client has everything they need to take ownership — credentials, documentation, training, and formal sign-off.

## Tasks
- [ ] Prepare credentials and access handoff document
  - [ ] Hosting/server access
  - [ ] Domain registrar access
  - [ ] Database credentials
  - [ ] Third-party service accounts (analytics, email, CDN, etc.)
- [ ] Write user-facing documentation (CMS guide, admin panel walkthrough)
- [ ] Record training video or conduct live walkthrough session
- [ ] Run /client-report handoff to generate handoff documentation
- [ ] Deliver final /site-audit report
- [ ] Get client sign-off on deliverables
- [ ] Set up ongoing maintenance agreement if applicable
- [ ] Archive project and update hub status

## Notes
Use /client-report handoff to generate professional handoff docs. Use /onboard-client if transitioning to a retainer/maintenance relationship.
